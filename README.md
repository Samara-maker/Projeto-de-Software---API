Sobre o Projeto:

O sistema WASH 1.0 é uma aplicação desenvolvida com o objetivo de gerenciar e automatizar processos de empresas do setor de limpeza.

A solução foi construída utilizando uma API baseada em arquitetura REST, responsável por centralizar as regras de negócio e o acesso aos dados. O sistema permite o cadastro, consulta, atualização e exclusão de informações, garantindo maior organização e controle das atividades.

Entre as principais funcionalidades, destacam-se:
- Cadastro e gerenciamento de clientes
- Controle de funcionários
- Organização de equipes
- Cadastro de serviços e categorias
- Agendamento de serviços

A aplicação foi desenvolvida utilizando Node.js com o framework NestJS, TypeScript e EJS para renderização de páginas, seguindo uma estrutura modular com controllers, services e entities.

O projeto faz parte da disciplina de Programação com Acesso a Banco de Dados do IFRO – Campus Ji-Paraná.
